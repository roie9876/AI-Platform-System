**1. Create `backend/app/repositories/token_log_repository.py`:**

```python
class TokenLogRepository(CosmosRepository):
    def __init__(self) -> None:
        super().__init__("token_logs")

    async def log_usage(self, tenant_id: str, log: dict) -> dict:
        return await self.create(tenant_id, log)

    async def get_usage_by_date_range(self, tenant_id: str, start: str, end: str) -> list[dict]:
        return await self.query(
            tenant_id,
            "SELECT * FROM c WHERE c.tenant_id = @tid AND c.timestamp >= @start AND c.timestamp <= @end ORDER BY c.timestamp DESC",
            [{"name": "@tid", "value": tenant_id}, {"name": "@start", "value": start}, {"name": "@end", "value": end}],
        )

    async def get_tenant_total_tokens(self, tenant_id: str, start: str, end: str) -> int:
        result = await self.query(
            tenant_id,
            "SELECT VALUE SUM(c.total_tokens) FROM c WHERE c.tenant_id = @tid AND c.timestamp >= @start AND c.timestamp <= @end",
            [{"name": "@tid", "value": tenant_id}, {"name": "@start", "value": start}, {"name": "@end", "value": end}],
        )
        return result[0] if result and result[0] else 0
```

**3. Create `backend/microservices/llm_proxy/main.py`:**

FastAPI app with these key components:

- **Config**: Read `AZURE_OPENAI_BASE` from env (default: `https://ai-platform-system.openai.azure.com`). Read `PROXY_PORT` from env (default: `8080`).
- **Lifespan**: Call `configure_logging(service_name="llm-proxy")`, `init_telemetry(service_name="llm-proxy")`, close cosmos client on shutdown. Pattern: copy from `mcp_proxy/main.py`.
- **No middleware**: This proxy does NOT use TenantMiddleware or CORSMiddleware — it's an internal-only service. Tenant identification comes from the URL path.

- **Route: `POST /proxy/{tenant_id}/{agent_id}/{path:path}`** (the main proxy endpoint per D-01):
  1. Extract `tenant_id` and `agent_id` from path params.
  2. Read request body as JSON.
  3. Determine if streaming: `body.get("stream", False)`.
  4. If streaming, inject `stream_options.include_usage = True`: `body.setdefault("stream_options", {})["include_usage"] = True`.
  5. Build upstream URL: `f"{AZURE_OPENAI_BASE}/{path}"`. Preserve query string (e.g., `api-version=...`).
  6. Forward all headers except `host`. Keep `api-key`, `content-type`, etc.
  7. Record `start_time = time.monotonic()`.
  8. If streaming: call `_handle_streaming()`. If not: call `_handle_non_streaming()`.

- **`_handle_non_streaming()`**:
  1. `response = await client.post(url, content=json.dumps(body), headers=headers)`.
  2. Parse response JSON.
  3. Extract `usage` from response body (if present).
  4. Calculate `latency_ms = (time.monotonic() - start_time) * 1000`.
  5. Fire-and-forget: `asyncio.create_task(_log_usage(tenant_id, agent_id, usage, model, api_type, False, latency_ms, response.status_code))`.
  6. Return response with same status code and content-type.

- **`_handle_streaming()`**:
  1. Use `client.stream("POST", url, content=json.dumps(body), headers=headers)` context manager.
  2. Define async generator that iterates `response.aiter_lines()`.
  3. For each SSE line starting with `data: ` (not `data: [DONE]`): parse JSON, check if `data.get("usage")` exists. If so, capture it as `usage_data`.
  4. Yield every line unchanged (including the usage chunk — transparent to client).
  5. After generator completes, fire `asyncio.create_task(_log_usage(...))` with captured usage.
  6. Return `StreamingResponse(generate(), media_type="text/event-stream", headers={"Cache-Control": "no-cache", "X-Accel-Buffering": "no"})`.

- **`_log_usage()` async function**:
  1. Build token log document:
     ```python
     {
         "tenant_id": tenant_id,
         "agent_id": agent_id,
         "timestamp": datetime.now(timezone.utc).isoformat(),
         "model": model or "unknown",
         "api": api_type,  # "chat-completions" or "responses" or "other"
         "stream": is_stream,
         "prompt_tokens": usage.get("prompt_tokens", 0) if usage else 0,
         "completion_tokens": usage.get("completion_tokens", 0) if usage else 0,
         "total_tokens": usage.get("total_tokens", 0) if usage else 0,
         "reasoning_tokens": (usage.get("completion_tokens_details", {}) or {}).get("reasoning_tokens", 0) if usage else 0,
         "cached_tokens": (usage.get("prompt_tokens_details", {}) or {}).get("cached_tokens", 0) if usage else 0,
         "latency_ms": round(latency_ms, 1),
         "status": status_code,
     }
     ```
  2. Call `await token_log_repo.log_usage(tenant_id, log_doc)`.
  3. Wrap in try/except — log error but never fail the response. Token logging is best-effort.

- **`_detect_api_type(path: str) -> str`**: Return `"chat-completions"` if path contains `chat/completions`, `"responses"` if path contains `responses`, else `"other"`.

- **`_extract_model(body: dict, response_data: dict | None) -> str`**: Return `body.get("model")` or `response_data.get("model")` or `"unknown"`.

- **Wildcard fallback route: `@app.api_route("/proxy/{tenant_id}/{agent_id}/{path:path}", methods=["GET", "PUT", "DELETE", "PATCH"])`**: Forward non-POST requests transparently without token extraction (passthrough per D-08).

- **Health routes**: Include `health_router` from `app.health`.

- **httpx client**: Create at module level with `timeout=httpx.Timeout(120.0, connect=10.0)`. Close in lifespan shutdown.

Port: The uvicorn CMD in Dockerfile uses port 8080 (not 8000). The `containerPort` in K8s deployment will also be 8080.
