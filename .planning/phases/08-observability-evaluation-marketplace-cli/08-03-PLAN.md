  <action>
    1. Create `backend/app/services/marketplace_service.py` with class `MarketplaceService`:

    **Agent Template Methods:**

    `async def list_agent_templates(db, category=None, search=None, featured_only=False, limit=50, offset=0)`:
    - Query agent_templates where is_public=True
    - Filter by category, search in name/description, featured_only
    - Return paginated list

    `async def get_agent_template(db, template_id)`:
    - Return single template with full details

    `async def publish_agent_template(db, agent_id, tenant_id, name, description, category, tags)`:
    - Load agent by id, verify tenant owns it
    - Create AgentTemplate from agent's config (system_prompt, settings, tool schemas)
    - Set author_tenant_id, is_public=True
    - Return template

    `async def import_agent_template(db, template_id, tenant_id)`:
    - Load template
    - Create new Agent in tenant with template's system_prompt and config
    - Increment template's install_count
    - Return new agent_id

    **Tool Template Methods:**

    `async def list_tool_templates(db, category=None, search=None, limit=50, offset=0)`:
    `async def get_tool_template(db, template_id)`:
    `async def publish_tool_template(db, tool_id, tenant_id, category, tags)`:
    `async def import_tool_template(db, template_id, tenant_id)`:
    - Same pattern: load template, create Tool in tenant, increment install_count

    2. Create `backend/app/api/v1/marketplace.py`:

    ```python
    router = APIRouter()

    # Agent Templates
    @router.get("/agents")  # Browse agent templates
    @router.get("/agents/{template_id}")  # Get template detail
    @router.post("/agents/publish")  # Publish agent as template {agent_id, name, description, category, tags}
    @router.post("/agents/{template_id}/import")  # Import template → creates agent

    # Tool Templates
    @router.get("/tools")  # Browse tool templates
    @router.get("/tools/{template_id}")
    @router.post("/tools/publish")  # Publish tool as template
    @router.post("/tools/{template_id}/import")  # Import template → creates tool
    ```

    3. Add schemas:
    - AgentTemplateResponse: id, name, description, category, tags, icon_name, author_name, install_count, version, is_featured, created_at
    - AgentTemplateDetailResponse: extends with system_prompt, config, tools_config
    - PublishAgentTemplateRequest: agent_id, name, description, category, tags
    - ToolTemplateResponse: id, name, description, category, tags, tool_type, install_count, version
    - PublishToolTemplateRequest: tool_id, name, description, category, tags

    4. Register in router.py:
    ```python
    from app.api.v1.marketplace import router as marketplace_router
    api_router.include_router(marketplace_router, prefix="/marketplace", tags=["marketplace"])
    ```
