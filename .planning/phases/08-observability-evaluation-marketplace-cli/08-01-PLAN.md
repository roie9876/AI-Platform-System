    1. Create `backend/app/services/observability_service.py` with class `ObservabilityService`:

    **Methods:**

    `async def get_dashboard_summary(db, tenant_id, time_range="7d", agent_id=None)`:
    - Query execution_logs joined with threads→agents, filtered by tenant_id
    - Time range parsing: "1h", "24h", "7d", "30d" → datetime offset from now
    - Return dict with:
      - total_requests: count of execution_logs
      - total_input_tokens: sum of token_count->'input_tokens'
      - total_output_tokens: sum of token_count->'output_tokens'
      - total_tokens: input + output
      - total_cost: calculated using model_pricing lookup (join state_snapshot->>'model_name' to model_pricing.model_name)
      - avg_latency_ms: avg(duration_ms)
      - p50_latency_ms, p95_latency_ms: percentile calculations
      - success_count: count where event_type != 'error'
      - error_count: count where event_type == 'error'
      - requests_per_minute: total_requests / time_range_minutes

    `async def get_token_usage_over_time(db, tenant_id, time_range, granularity, agent_id=None, model_name=None)`:
    - Group by time bucket (using date_trunc or generate_series)
    - Return list of {time, input_tokens, output_tokens, total_tokens}

    `async def get_cost_breakdown(db, tenant_id, time_range, group_by="agent")`:
    - group_by options: "agent", "model", "tenant"
    - Join execution_logs → threads → agents for agent grouping
    - Join state_snapshot->>'model_name' → model_pricing for cost calc
    - Return list of {name, total_tokens, total_cost, request_count}

    `async def get_top_agents(db, tenant_id, time_range, limit=10)`:
    - Top agents by token usage or cost
    - Return list of {agent_id, agent_name, total_tokens, total_cost, request_count}

    `async def check_cost_alerts(db, tenant_id)`:
    - For each active CostAlert where scope matches tenant
    - Calculate current period spend
    - If threshold exceeded and not recently triggered (>1h), return alert
    - Update last_triggered_at

    `async def get_execution_logs(db, tenant_id, agent_id=None, limit=50, offset=0)`:
    - Paginated list of execution logs with thread/agent info
    - Return list with event_type, duration_ms, token_count, model_name, created_at
