  <action>
    1. Create `backend/app/services/evaluation_service.py` with class `EvaluationService`:

    **Methods:**

    `async def run_evaluation(db, test_suite_id, tenant_id)`:
    - Load test suite + all test cases
    - Create EvaluationRun with status="running"
    - For each test case:
      - Create a temporary thread for the agent
      - Call AgentExecutionService.execute() with the test case input_message
      - Capture response, latency, tokens
      - Compute metrics:
        - similarity_score: simple word overlap / Jaccard similarity between actual_output and expected_output (if expected_output exists)
        - keyword_match_rate: fraction of expected_keywords found in actual_output
        - latency_ms: from execution log
        - input_tokens, output_tokens: from execution log
      - Determine pass/fail: score >= 0.5 = passed
      - Create EvaluationResult row
      - Clean up temporary thread
    - Update EvaluationRun: status="completed", summary with aggregated stats
    - Return run_id

    `async def compare_runs(db, run_ids: list[UUID])`:
    - Load multiple EvaluationRuns with their results
    - Return comparison: per-test-case diff, aggregate score diff, latency diff
    - Structure: {runs: [{run_id, config_version, summary}], per_case: [{test_case_id, input, results_by_run}]}

    `async def get_run_results(db, run_id)`:
    - Return all EvaluationResults for a run with test case details

    2. Create `backend/app/api/v1/evaluations.py`:

    ```python
    # Test Suite CRUD
    @router.get("/test-suites")  # List test suites (filtered by agent_id optional)
    @router.post("/test-suites")  # Create test suite
    @router.get("/test-suites/{suite_id}")  # Get suite with cases
    @router.put("/test-suites/{suite_id}")
    @router.delete("/test-suites/{suite_id}")

    # Test Cases within a suite
    @router.post("/test-suites/{suite_id}/cases")  # Add test case
    @router.put("/test-suites/{suite_id}/cases/{case_id}")
    @router.delete("/test-suites/{suite_id}/cases/{case_id}")

    # Evaluation Runs
    @router.post("/test-suites/{suite_id}/run")  # Trigger evaluation run
    @router.get("/runs")  # List runs (filtered by agent_id, suite_id)
    @router.get("/runs/{run_id}")  # Get run with results
    @router.get("/runs/{run_id}/results")  # Get detailed results

    # Comparison
    @router.post("/compare")  # POST {run_ids: [id1, id2]} → comparison data
    ```

    - EvaluationResultResponse: id, test_case_id, actual_output, score, metrics, status
    - ComparisonResponse: runs list, per_case comparison
