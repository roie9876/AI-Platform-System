<objective>
Refactor AgentExecutionService and WorkflowEngine to use HTTP-based inter-service communication instead of direct Python imports, breaking the monolith's tight coupling between execution, tool, and workflow subsystems.

Purpose: After Plan 20-01 creates the microservice scaffolding, the service classes still import each other directly. This plan replaces those direct imports with HTTP calls via ServiceClient, enabling each microservice to run in its own container without importing code from other services.

Output: AgentExecutionService and WorkflowEngine use ServiceClient for cross-service communication; agent-executor exposes internal endpoint for workflow invocation.

**2. Add internal endpoint to `backend/microservices/agent_executor/main.py`:**

Add an internal router for workflow-engine to call:

```python
from fastapi import APIRouter, Request
from app.services.agent_execution import AgentExecutionService

internal_router = APIRouter(prefix="/api/v1/internal", tags=["internal"])

@internal_router.post("/agents/{agent_id}/execute")
async def internal_execute_agent(agent_id: str, request: Request):
    """Internal endpoint for workflow-engine to invoke agent execution."""
    body = await request.json()
    tenant_id = request.state.tenant_id
    
    service = AgentExecutionService()
    # Collect streaming response into single result
    response_chunks = []
    async for chunk in service.execute_stream(
        agent_id=agent_id,
        message=body["message"],
        tenant_id=tenant_id,
        user_id=body.get("user_id", request.state.user_id),
        thread_id=body.get("thread_id"),
        auth_token=request.headers.get("Authorization", "").replace("Bearer ", ""),
    ):
        if isinstance(chunk, str):
            response_chunks.append(chunk)
    
    return {"response": "".join(response_chunks), "agent_id": agent_id}

app.include_router(internal_router)
```

This breaks the circular dependency: workflow-engine calls agent-executor over HTTP, never importing AgentExecutionService directly.
