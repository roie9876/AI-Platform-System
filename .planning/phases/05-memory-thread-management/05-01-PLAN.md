     class Thread(Base, UUIDMixin, TimestampMixin):
         __tablename__ = "threads"
         title = Column(String(255), nullable=True)  # Auto-generated or user-set
         agent_id = Column(UUID, ForeignKey("agents.id", ondelete="CASCADE"), nullable=False, index=True)
         user_id = Column(UUID, ForeignKey("users.id", ondelete="CASCADE"), nullable=False, index=True)
         tenant_id = Column(UUID, ForeignKey("tenants.id"), nullable=False, index=True)
         is_active = Column(Boolean, default=True, nullable=False)

     class ThreadMessage(Base, UUIDMixin, TimestampMixin):
         __tablename__ = "thread_messages"

         role = Column(String(20), nullable=False)  # user, assistant, system, tool
         content = Column(Text, nullable=False)
         message_metadata = Column(JSONB, nullable=True)  # tool_calls, sources, etc.

     ```python
     from pgvector.sqlalchemy import Vector

     class AgentMemory(Base, UUIDMixin, TimestampMixin):
         __tablename__ = "agent_memories"
         agent_id = Column(UUID, ForeignKey("agents.id", ondelete="CASCADE"), nullable=False, index=True)
         user_id = Column(UUID, ForeignKey("users.id", ondelete="CASCADE"), nullable=False, index=True)
         tenant_id = Column(UUID, ForeignKey("tenants.id"), nullable=False, index=True)
         content = Column(Text, nullable=False)  # The memory text
         embedding = Column(Vector(1536), nullable=True)  # text-embedding-3-small dimension
         memory_type = Column(String(50), default="knowledge", nullable=False)  # knowledge, preference, fact
         source_thread_id = Column(UUID, ForeignKey("threads.id", ondelete="SET NULL"), nullable=True)
     ```

  1. Add Pydantic schemas to `schemas.py`:
     ```python
     class ThreadCreateRequest(BaseModel):
         agent_id: UUID
         title: Optional[str] = None

     class ThreadResponse(BaseModel):
         id: UUID
         title: Optional[str]
         agent_id: UUID
         user_id: UUID
         tenant_id: UUID
         is_active: bool
         created_at: datetime
         updated_at: datetime
         message_count: int = 0  # Computed field
         last_message_preview: Optional[str] = None
         model_config = {"from_attributes": True}

     class ThreadListResponse(BaseModel):
         threads: List[ThreadResponse]
         total: int

     class ThreadMessageResponse(BaseModel):
         id: UUID
         thread_id: UUID
         role: str
         content: str
         message_metadata: Optional[dict] = None
         sequence_number: int
         created_at: datetime
         model_config = {"from_attributes": True}

     class ThreadMessagesResponse(BaseModel):
         messages: List[ThreadMessageResponse]
         total: int
     ```

  2. Create `backend/app/api/v1/threads.py` with endpoints:
     - `POST /threads` — Create a new thread for an agent. Validates agent exists and belongs to tenant. Sets user_id from current_user.
     - `GET /threads?agent_id={id}` — List threads for an agent (filtered by tenant_id and user_id). Returns newest first. Include message_count and last_message_preview (first 100 chars of last assistant message).
     - `GET /threads/{thread_id}` — Get single thread with validation (tenant + user ownership).
     - `GET /threads/{thread_id}/messages` — Get all messages for a thread, ordered by sequence_number ASC.
     - `DELETE /threads/{thread_id}` — Soft-delete (set is_active=false) or hard-delete. Use hard delete for simplicity in PoC.
     - `PATCH /threads/{thread_id}` — Update thread title.

     All endpoints use `get_current_user` and `get_tenant_id` dependencies. Threads are scoped: user can only see their own threads within their tenant.

  3. Register in `router.py`:
     ```python
     from app.api.v1.threads import router as threads_router
     api_router.include_router(threads_router, prefix="/threads", tags=["threads"])
     ```
