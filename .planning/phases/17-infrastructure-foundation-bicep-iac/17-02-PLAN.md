Use a Bicep array variable `containerNames` holding all 35 container names, then use a `for` loop to create containers programmatically:

```bicep
var containerNames = [
  'agents'
  'agent_config_versions'
  // ... all 35
]

resource containers 'Microsoft.DocumentDB/databaseAccounts/sqlDatabases/containers@2024-05-15' = [for name in containerNames: {
  parent: database
  name: name
  properties: {
    resource: {
      id: name
      partitionKey: {
        paths: ['/tenant_id']
        kind: 'Hash'
      }
    }
  }
}]
```
