Create three files:

**1. `frontend/src/contexts/tenant-context.tsx`:**
- "use client" directive
- Interface `Tenant` matching TenantResponse: `{ id: string; name: string; slug: string; admin_email: string; status: string; settings: Record<string, unknown>; created_at: string; updated_at: string }`
- Interface `TenantContextType`: `{ tenants: Tenant[]; selectedTenantId: string | null; setSelectedTenantId: (id: string) => void; selectedTenant: Tenant | null; loading: boolean; refreshTenants: () => void }`
- `TenantProvider` component:
  - Uses `useAuth()` to check if user has `platform_admin` in `user.roles`
  - If platform_admin: fetches `GET /api/v1/tenants` via `apiFetch<{ tenants: Tenant[] }>("/api/v1/tenants")` on mount
  - Stores tenants list and selectedTenantId in useState
  - Default selectedTenantId = user?.tenant_id (from auth context)
  - selectedTenant = derived via `tenants.find(t => t.id === selectedTenantId) ?? null`
  - refreshTenants function re-fetches tenant list
  - If NOT platform_admin: tenants=[], selectedTenantId=user?.tenant_id, loading=false
- Export `useTenant()` hook with context validation

**2. `frontend/src/components/layout/tenant-selector.tsx`:**
- "use client" directive
- Import `useTenant` from contexts/tenant-context
- Import `useAuth` from contexts/auth-context

- Render: `<div className="flex items-center gap-2">` with label `<span className="text-xs text-gray-500">Tenant:</span>` + `<select>` dropdown
- Select classes: `rounded-md border border-gray-200 px-3 py-1.5 text-sm text-gray-700 bg-white min-w-[200px] focus:border-blue-500 focus:ring-1 focus:ring-blue-500`
- Option for each tenant: `<option value={tenant.id}>{tenant.name}</option>`
- onChange calls `setSelectedTenantId(e.target.value)`
