    2. Create agent_mcp_tools.py with 3 endpoints following the existing agent_tools pattern:

    - POST /{agent_id}/mcp-tools — attach MCP tool to agent
      Body: AgentMCPToolAttachRequest { mcp_tool_id: UUID }
      Validation: agent exists + tenant match, mcp_tool exists + tenant match, no duplicate (409)
      Returns: AgentMCPToolResponse with joined tool+server data (201)

    - DELETE /{agent_id}/mcp-tools/{mcp_tool_id} — detach MCP tool from agent
      Validation: agent exists + tenant, attachment exists (404 if not found)
      Returns: 204

    - GET /{agent_id}/mcp-tools — list MCP tools attached to agent
      Join AgentMCPTool → MCPDiscoveredTool → MCPServer to get tool_name, description, server_name, is_available
      Returns: list[AgentMCPToolResponse]

    Use router = APIRouter() (separate router for agent MCP tools).
    Import AgentMCPTool, MCPDiscoveredTool, MCPServer, Agent models.
    Follow exact same auth/tenant pattern as existing agent_tools_router in tools.py.
