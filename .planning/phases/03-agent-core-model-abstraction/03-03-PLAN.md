<!-- API contracts from Plan 03-01 -->

Agent API (GET /api/v1/agents):
```json
{
  "agents": [
    {
      "id": "uuid", "name": "str", "description": "str|null",
      "system_prompt": "str|null", "status": "inactive|active|error",
      "temperature": 0.7, "max_tokens": 1024, "timeout_seconds": 30,
      "model_endpoint_id": "uuid|null", "current_config_version": 1,
      "tenant_id": "uuid", "created_at": "iso", "updated_at": "iso"
    }
  ],
  "total": 1
}
```

POST /api/v1/agents (create):
```json
{ "name": "str", "description": "str|null", "system_prompt": "str|null",
  "model_endpoint_id": "uuid|null", "temperature": 0.7,
  "max_tokens": 1024, "timeout_seconds": 30 }
```

ModelEndpoint API (GET /api/v1/model-endpoints):
```json
{
  "endpoints": [
    {
      "id": "uuid", "name": "str", "provider_type": "azure_openai|openai|anthropic|custom",
      "endpoint_url": "str|null", "model_name": "str", "auth_type": "entra_id|api_key",
      "is_active": true, "priority": 0,
      "tenant_id": "uuid", "created_at": "iso", "updated_at": "iso"
    }
  ],
  "total": 1
}
```

GET /api/v1/agents/{id}/versions:
```json
[
  { "id": "uuid", "agent_id": "uuid", "version_number": 1,
    "config_snapshot": {...}, "change_description": "str|null", "created_at": "iso" }
]
```

POST /api/v1/agents/{id}/rollback/{version_number} → returns updated AgentResponse
