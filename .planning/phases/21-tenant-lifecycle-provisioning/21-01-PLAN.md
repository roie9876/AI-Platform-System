**TenantService** (backend/app/services/tenant_service.py):
- Create `TenantService` class wrapping `TenantRepository`.
- **Lifecycle states**: Define valid states as a module-level dict:
  ```
  TENANT_STATES = ["provisioning", "active", "suspended", "deactivated", "deleted"]
  ```
- **Valid transitions** as a dict:
  ```
  VALID_TRANSITIONS = {
      "provisioning": ["active", "deleted"],
      "active": ["suspended", "deactivated"],
      "suspended": ["active", "deactivated"],
      "deactivated": ["deleted"],
      "deleted": [],
  }
  ```
- `create_tenant(name, slug, admin_email, settings=None)` → validate slug uniqueness via `get_by_slug`, create tenant doc with:
  - `id`: uuid4
  - `tenant_id`: same as id (self-partitioned)
  - `name`, `slug`, `admin_email`
  - `status`: "provisioning"
  - `settings`: `{"display_name": name, "allowed_providers": [], "token_quota": 100000, "feature_flags": {}}`
  - `created_at`, `updated_at` timestamps
  - Return created tenant dict.
  - Raise `ValueError` if slug already exists.

- `get_tenant(tenant_id)` → get from repo.
- `list_tenants(status=None)` → if status provided use `get_by_status`, else `list_all_tenants`.
- `update_tenant(tenant_id, updates: dict)` → get tenant, merge updates (only allowed fields: name, admin_email), save via upsert.
- `transition_state(tenant_id, new_state)` → get tenant, validate transition against `VALID_TRANSITIONS`, update status field, return updated tenant. Raise `ValueError` with descriptive message for invalid transitions.
- `update_settings(tenant_id, settings: dict)` → get tenant, merge settings dict into tenant["settings"], upsert. Allowed setting keys: display_name, allowed_providers, token_quota, feature_flags.
- `delete_tenant(tenant_id)` → transition to "deleted" state (soft delete, do NOT remove from DB). If not in deactivated state, raise ValueError.

**Tenant schemas** (append to backend/app/api/v1/schemas.py):
```python
class TenantCreateRequest(BaseModel):
    name: str = Field(..., min_length=1, max_length=255)
    slug: str = Field(..., min_length=1, max_length=100, pattern=r"^[a-z0-9-]+$")
    admin_email: EmailStr

class TenantUpdateRequest(BaseModel):
    name: Optional[str] = Field(default=None, min_length=1, max_length=255)
    admin_email: Optional[EmailStr] = None

class TenantSettingsUpdateRequest(BaseModel):
    display_name: Optional[str] = Field(default=None, max_length=255)
    allowed_providers: Optional[List[str]] = None
    token_quota: Optional[int] = Field(default=None, ge=0)
    feature_flags: Optional[dict] = None

class TenantStateTransitionRequest(BaseModel):
    state: str = Field(..., pattern=r"^(active|suspended|deactivated|deleted)$")

class TenantResponse(BaseModel):
    id: str
    name: str
    slug: str
    admin_email: str
    status: str
    settings: dict
    created_at: str
    updated_at: str

class TenantListResponse(BaseModel):
    tenants: List[TenantResponse]
```

**Tenant API** (backend/app/api/v1/tenants.py):
- `POST /` — create tenant. Body: TenantCreateRequest. Validate calling user has platform admin role (check `request.state.user_context["roles"]` contains "platform_admin"). Return 201 + TenantResponse. Catch ValueError → 409 Conflict.
- `GET /` — list tenants. Optional query param `status`. Return TenantListResponse. Requires platform_admin role.
- `GET /{tenant_id}` — get single tenant. Return TenantResponse. Requires platform_admin role.
- `PATCH /{tenant_id}` — update tenant fields. Body: TenantUpdateRequest. Return TenantResponse. Requires platform_admin role.
- `PATCH /{tenant_id}/state` — transition lifecycle state. Body: TenantStateTransitionRequest. Return TenantResponse. Catch ValueError → 400 Bad Request. Requires platform_admin role.
- `PATCH /{tenant_id}/settings` — update tenant settings. Body: TenantSettingsUpdateRequest. Return TenantResponse. Requires platform_admin role.
- `DELETE /{tenant_id}` — soft delete (transitions to "deleted"). Return 204. Catch ValueError → 400 Bad Request. Requires platform_admin role.

**Role check helper** — create a `require_platform_admin(request: Request)` dependency that raises 403 if user doesn't have "platform_admin" role.
