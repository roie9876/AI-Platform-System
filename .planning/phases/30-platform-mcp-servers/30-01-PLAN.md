    **`embedding.py`** â€” Azure OpenAI embedding client with query cache (per D-08, D-05):
    - Class `EmbeddingService` initialized with `AsyncAzureOpenAI` client
    - Method `async embed_text(text: str) -> list[float]`: calls `client.embeddings.create(model="text-embedding-3-small", input=[text])`, returns `response.data[0].embedding`
    - Query cache: in-memory dict `{sha256(text): embedding_vector}` with max 1000 entries and LRU eviction. The cache is for the lifetime of the pod â€” not persisted to Cosmos (D-05 `memory_query_cache` container is for cross-pod cache, implement in a later enhancement)
    - On error: log warning, return empty list

    **`memory.py`** â€” Memory MCP tools (per D-01, D-04, D-06):
    - Import `get_cosmos_container` from `app.repositories.cosmos_client`
    - `async memory_store(tenant_id: str, agent_id: str, content: str, memory_type: str = "knowledge", user_id: str = "", source: str = "") -> dict`:
      - Generate embedding via `EmbeddingService.embed_text(content)`
      - Create document: `{id: uuid4(), tenant_id, agent_id, user_id, content, memory_type, type: "memory", embedding, source, created_at: utcnow().isoformat(), updated_at: utcnow().isoformat()}`
      - Upsert to `agent_memories` container via `container.create_item(body=doc)`
      - Return `{id, content, memory_type, created_at}`
    - `async memory_search(tenant_id: str, agent_id: str, query: str, top_k: int = 5, memory_type: str = "") -> list[dict]`:
      - Embed query via `EmbeddingService.embed_text(query)`
      - If embedding empty, return error dict
      - Build SQL: `SELECT TOP @top_k c.id, c.content, c.memory_type, c.agent_id, c.user_id, c.created_at, VectorDistance(c.embedding, @query_vector) AS similarity_score FROM c WHERE c.tenant_id = @tenant_id AND c.agent_id = @agent_id ORDER BY VectorDistance(c.embedding, @query_vector)`
      - If `memory_type` provided, add `AND c.memory_type = @memory_type` to WHERE clause
      - Execute via `container.query_items(query=sql, parameters=[...], partition_key=tenant_id)`
      - Return list of result dicts (without embedding field â€” strip it out to reduce token usage)
    - `async memory_store_structured(tenant_id: str, agent_id: str, key: str, value: str, category: str = "preference") -> dict`:
      - Create document: `{id: sha256(f"{tenant_id}:{agent_id}:{key}"), tenant_id, agent_id, key, value, category, type: "structured", created_at, updated_at}`
      - Upsert to `structured_memories` container
      - Return `{key, value, category, updated_at}`
    - `async memory_get_structured(tenant_id: str, agent_id: str, key: str = "", category: str = "") -> list[dict]`:
      - If key provided: query by exact key match
      - If category provided: query by category
      - If neither: list all structured memories for agent
      - Return list of `{key, value, category, updated_at}`
