    - Test 3: get_agent_config(tenant_id, agent_id) returns agent name, description, system_prompt, model_name
    - Test 4: list_configured_groups(tenant_id, agent_id) returns all configured groups with their settings
  </behavior>
  <action>
    Create `backend/microservices/mcp_platform_tools/platform_config.py`:

    - Import `get_cosmos_container` from `app.repositories.cosmos_client`

    - `async def get_group_instructions(tenant_id: str, agent_id: str, group_jid: str) -> dict`:
      - Query `agents` container: `SELECT * FROM c WHERE c.tenant_id = @tid AND c.id = @aid` with partition_key=tenant_id
      - Extract `openclaw_config.whatsapp.groups[group_jid]` from agent document
      - Return `{group_jid, system_prompt: group_config.get("systemPrompt", ""), require_mention: group_config.get("requireMention", true), contact_policy: group_config.get("contactPolicy", "open"), found: true}`
      - If group_jid not in groups dict: return `{group_jid, system_prompt: "", found: false}`
      - If agent not found: return `{error: "Agent not found"}`

    - `async def get_agent_config(tenant_id: str, agent_id: str) -> dict`:
      - Read agent document from `agents` container by id with partition_key=tenant_id
      - Return `{id, name, description, system_prompt, model_name, created_at}` — strip sensitive fields (openclaw_config, api keys)
      - If agent not found: return `{error: "Agent not found"}`

    - `async def list_configured_groups(tenant_id: str, agent_id: str) -> dict`:
      - Read agent document, extract `openclaw_config.whatsapp.groups`
      - For each group: return `{group_jid, system_prompt (first 100 chars), require_mention, contact_policy}`
      - If no groups configured: return `{groups: [], count: 0}`
      - Return `{groups: [...], count: N}`

    Per D-15, D-16, D-17 — read directly from Cosmos `agents` container. No K8s API dependency. Same data source as openclaw_service.py.
  </action>
