**In frontend/src/lib/api.ts:**

1. Add module-level tenant ID state at the top of the file (after imports):
```typescript
let _currentTenantId: string | null = null;

export function setCurrentTenantId(id: string | null) {
  _currentTenantId = id;
}
```

2. Inside the `apiFetch` function, after building the `headers` object and after the Authorization header block, add tenant header injection:
```typescript
if (_currentTenantId) {
  headers["X-Tenant-Id"] = _currentTenantId;
}
```

This goes AFTER the `if (token)` block and BEFORE the `fetch()` call.

**In frontend/src/contexts/tenant-context.tsx:**

1. Add import at the top:
```typescript
import { setCurrentTenantId } from "@/lib/api";
```

2. Inside `TenantProvider`, add a useEffect that syncs selectedTenantId to the API module. Place it after the existing `useEffect` blocks (after the one that sets selectedTenantId from user.tenant_id):
```typescript
useEffect(() => {
  setCurrentTenantId(selectedTenantId);
}, [selectedTenantId]);
```
