**backend/app/api/v1/azure_connections.py:**
Create router for connection management. Endpoints:

1. `POST /azure/connections` — Create connection
   - Body: `AzureConnectionCreate` with: agent_id (UUID), azure_subscription_id (UUID), resource_type (str), resource_name (str), resource_id (str), endpoint (str, optional), region (str, optional), auth_type (str — "api_key" or "managed_identity"), credentials (str, optional, the raw key/connection string)
   - Encrypt `credentials` via `encrypt_api_key()` → store in `credentials_encrypted`
   - Validate agent_id belongs to tenant via `get_tenant_id()`
   - Set health_status = "unknown"
   - Return `AzureConnectionResponse`

2. `GET /azure/agents/{agent_id}/connections` — List connections for an agent
   - Filter AzureConnection by agent_id AND tenant_id
   - Return list of `AzureConnectionResponse`

3. `DELETE /azure/connections/{connection_id}` — Delete a connection
   - Delete by id AND tenant_id
   - Return 204

4. `PATCH /azure/connections/{connection_id}` — Update connection config
   - Body: `AzureConnectionUpdate` with: auth_type (optional), credentials (optional), config (optional JSONB)
   - If credentials provided, re-encrypt. Update other fields.
   - Return `AzureConnectionResponse`

5. `POST /azure/connections/{connection_id}/health-check` — Run health check
   - For now, set health_status = "connected" and last_health_check = utcnow()
   - Future: actually ping the resource endpoint
   - Return `AzureConnectionResponse`

**backend/app/api/v1/knowledge.py:**
Create router for knowledge/AI Search integration. Endpoints:

1. `GET /knowledge/connections/{connection_id}/indexes` — List AI Search indexes for a connection
   - Validate connection belongs to tenant
   - Validate connection.resource_type == "Microsoft.Search/searchServices"
   - Decrypt access_token from connection's Azure subscription
   - Call `AzureARMService.list_search_indexes(access_token, connection.resource_id)`
   - Return `SearchIndexListResponse` with list of `SearchIndex` (name, document_count if available)

2. `POST /knowledge/connections/{connection_id}/indexes` — Select indexes for RAG
   - Body: `SelectIndexesRequest` with: index_names (list[str])
   - Update connection.config to {"selected_indexes": index_names}
   - Return updated `AzureConnectionResponse`

3. `GET /knowledge/agents/{agent_id}/indexes` — List all selected indexes for an agent
   - Find all AzureConnections for agent_id where resource_type == "Microsoft.Search/searchServices" and config contains selected_indexes
   - Return `AgentKnowledgeResponse` with list of connections and their selected indexes

```python
async def retrieve_from_azure_search(
    self, query: str, agent_id: uuid.UUID, db: AsyncSession, top_k: int = 5
) -> list[dict]:
    """Retrieve from connected Azure AI Search indexes.
    1. Query AzureConnection for agent_id where resource_type == 'Microsoft.Search/searchServices' and config['selected_indexes'] is not empty
    2. For each connection with selected indexes:
       a. Decrypt credentials_encrypted (API key) from the connection
       b. Use azure.search.documents.SearchClient to search each selected index
       c. Collect results
    3. Return combined results sorted by score, limited to top_k
    """
```
