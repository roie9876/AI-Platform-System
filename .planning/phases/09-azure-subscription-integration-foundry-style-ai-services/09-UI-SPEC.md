## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding |
| sm | 8px | Compact element spacing, inline padding, badge margins |
| md | 16px | Default element spacing, card padding, form field gaps |
| lg | 24px | Section padding, sidebar item vertical rhythm |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Page header to content gap |
| 3xl | 64px | Not used in this phase |

Exceptions: Touch targets for sidebar nav items: 40px height minimum. Modal header: 56px height. Region badges: 6px vertical / 10px horizontal padding.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFFFFF | Page backgrounds, content areas, modal backgrounds |
| Secondary (30%) | #F9FAFB | Sidebar background, card surfaces, table header rows, section backgrounds, filter bar |
| Accent (10%) | #7C3AED | See accent reserved-for list below |
| Destructive | #DC2626 | Delete/disconnect buttons, error badges, error text |

### Accent Reserved-For (exhaustive list)

- Primary action buttons ("Connect subscription", "Add tool", "Connect", "Create agent")
- Active sidebar nav item indicator (left 3px border or background highlight)
- Selected tab underline (Configured / Catalog / Custom tabs)
- "Preview" badge background (at 10% opacity: `#7C3AED1A` with `#7C3AED` text)
- Toggle switches in "on" state
- Page header breadcrumb active segment

Accent is NOT used for: links (use `#2563EB` blue), hover states on non-primary elements, borders, icons in nav (icons use `#6B7280` gray-500 default, `#7C3AED` when active).

## Layout Specs

### Sidebar (Foundry-Style)

- Width: 240px (down from current 256px)
- Background: #FFFFFF (white — not dark)
- Right border: 1px solid #E5E7EB
- Nav item height: 40px
- Nav item padding: 12px horizontal
- Icon size: 20px (lucide default)
- Icon color: #6B7280 (gray-500) default, #7C3AED (accent) when active
- Active item: 3px left border in #7C3AED + #F5F3FF background (violet-50)
- Section dividers: 1px solid #E5E7EB with 16px vertical margin
- User section: bottom of sidebar, 1px top border

### Agent Config Split-Pane

- Left panel width: 380px, not resizable
- Left panel background: #FFFFFF
- Right panel: flex-1, #F9FAFB background
- Divider: 1px solid #E5E7EB
- Collapsible section header: 48px height, chevron on left, heading text, content below when expanded

### Tool Catalog Modal

- Width: 720px max, centered
- Overlay: rgba(0,0,0,0.4)
- Border radius: 12px
- Card grid: 2 columns on md+, gap 16px
- Tab bar: bottom border, selected tab has 2px bottom border in accent

### Cards (General Pattern)

- Background: #FFFFFF
- Border: 1px solid #E5E7EB
- Border radius: 8px
- Padding: 16px
- Hover: border-color transitions to #D1D5DB (gray-300)
- Shadow: none (flat design, matching Foundry)
