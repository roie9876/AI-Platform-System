**backend/app/api/v1/tools.py** — New file, `router = APIRouter()`:
Following the pattern from `agents.py` exactly (same imports: APIRouter, Depends, HTTPException, Request, select, func, AsyncSession, get_db, get_tenant_id, get_current_user):

Endpoints:
- `POST /` → create_tool: Create tool (tenant-scoped, is_platform_tool=False), return 201
- `GET /` → list_tools: List tools where `tenant_id == current_tenant OR is_platform_tool == True`, return ToolListResponse
- `GET /{tool_id}` → get_tool: Get single tool (check tenant_id match or is_platform_tool), 404 if not found
- `PUT /{tool_id}` → update_tool: Update tool (only if tenant-owned, not platform), 404/403 if not found/platform
- `DELETE /{tool_id}` → delete_tool: Delete tool (only if tenant-owned), return 204
- `POST /agents/{agent_id}/tools` → attach_tool: Create AgentTool record (validate both agent and tool exist in tenant, check for existing), return 201
- `DELETE /agents/{agent_id}/tools/{tool_id}` → detach_tool: Delete AgentTool record, return 204
- `GET /agents/{agent_id}/tools` → list_agent_tools: List tools attached to agent

Note: The attach/detach endpoints will be mounted under `/agents` prefix in router.py, so define them in a separate `agent_tools_router = APIRouter()`.

**backend/app/api/v1/data_sources.py** — New file, `router = APIRouter()`:
Same pattern as tools.py:
- `POST /` → create_data_source: Create data source. If `credentials` provided in request body, encrypt using `encrypt_api_key()` from secret_store and store in `credentials_encrypted`. Return 201.
- `GET /` → list_data_sources: List data sources for current tenant
- `GET /{data_source_id}` → get_data_source: Get single, 404 if not found or wrong tenant
- `PUT /{data_source_id}` → update_data_source: Update, 404 if not found
- `DELETE /{data_source_id}` → delete_data_source: Delete, return 204
- Agent attachment endpoints in separate router (same pattern as tools):
  - `POST /agents/{agent_id}/data-sources` → attach
  - `DELETE /agents/{agent_id}/data-sources/{data_source_id}` → detach
  - `GET /agents/{agent_id}/data-sources` → list
