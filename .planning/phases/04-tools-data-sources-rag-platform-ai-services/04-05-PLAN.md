<!-- Backend API contracts (from Plans 01, 04) -->

Tool CRUD API:
- GET /api/v1/tools → { tools: Tool[], total: number }
- POST /api/v1/tools → { name, description, input_schema, timeout_seconds? }
- DELETE /api/v1/tools/{tool_id}
- POST /api/v1/tools/{tool_id}/agents/{agent_id} → attach tool to agent
- DELETE /api/v1/tools/{tool_id}/agents/{agent_id} → detach tool from agent

DataSource CRUD API:
- GET /api/v1/data-sources → { data_sources: DataSource[], total: number }
- POST /api/v1/data-sources → { name, type: "file_upload"|"url", url? }
- DELETE /api/v1/data-sources/{data_source_id}
- POST /api/v1/data-sources/{data_source_id}/documents/upload → FormData with file
- POST /api/v1/data-sources/{data_source_id}/documents/ingest-url → { url }

AI Services API:
- GET /api/v1/ai-services?agent_id={id} → { tools: PlatformToolResponse[], total: number }
- POST /api/v1/ai-services/toggle?agent_id={id} → { tool_id, enabled: boolean }
