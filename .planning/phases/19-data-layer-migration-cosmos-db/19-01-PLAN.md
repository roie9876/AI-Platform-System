2. Add to `backend/app/core/config.py` two new settings:
   - `COSMOS_ENDPOINT: str = ""` (e.g. `https://stumsft-aiplatform-prod-cosmos.documents.azure.com:443/`)
   - `COSMOS_DATABASE: str = "aiplatform"`

3. Create `backend/app/repositories/cosmos_client.py`:
   - Import `CosmosClient` from `azure.cosmos.aio`
   - Import `DefaultAzureCredential` from `azure.identity.aio`
   - Import `settings` from `app.core.config`
   - Module-level `_client: CosmosClient | None = None`
   - `async def get_cosmos_client() -> CosmosClient` — lazy singleton, creates CosmosClient with `DefaultAzureCredential()` and `settings.COSMOS_ENDPOINT`
   - `async def get_cosmos_container(container_name: str) -> ContainerProxy` — gets database client for `settings.COSMOS_DATABASE`, then container client
   - `async def close_cosmos_client()` — cleanup for shutdown

4. Create `backend/app/repositories/base.py`:
   - `class CosmosRepository`:
     - `__init__(self, container_name: str)` — stores container name
     - `async def _container(self) -> ContainerProxy` — calls `get_cosmos_container(self.container_name)`
     - `async def create(self, tenant_id: str, item: dict) -> dict` — sets `item["tenant_id"] = tenant_id`, `item["id"] = item.get("id", str(uuid4()))`, adds `created_at` and `updated_at` as ISO 8601 strings, calls `container.create_item(item)`
     - `async def get(self, tenant_id: str, item_id: str) -> dict | None` — `container.read_item(item=item_id, partition_key=tenant_id)`, catches `CosmosResourceNotFoundError` returns None
     - `async def query(self, tenant_id: str, query: str, parameters: list | None = None) -> list[dict]` — `container.query_items(query=query, parameters=parameters, partition_key=tenant_id)`, async iteration to list
     - `async def list_all(self, tenant_id: str) -> list[dict]` — shorthand for `SELECT * FROM c WHERE c.tenant_id = @tid`
     - `async def update(self, tenant_id: str, item_id: str, item: dict, etag: str | None = None) -> dict` — sets `item["updated_at"]` to now ISO 8601. If etag provided, uses `replace_item` with `etag=etag, match_condition=MatchConditions.IfNotModified`. Otherwise plain `replace_item`.
     - `async def delete(self, tenant_id: str, item_id: str) -> None` — `container.delete_item(item=item_id, partition_key=tenant_id)`
     - `async def upsert(self, tenant_id: str, item: dict) -> dict` — `container.upsert_item(item)`
     - `async def count(self, tenant_id: str, filter_clause: str = "", parameters: list | None = None) -> int` — `SELECT VALUE COUNT(1) FROM c WHERE c.tenant_id = @tid {filter_clause}`

5. Create `backend/app/repositories/__init__.py` with imports of `CosmosRepository` and `get_cosmos_container`.
